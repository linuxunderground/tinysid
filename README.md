# TinySID

## Overview
TinySID is the world's smallest SID-Player written by
Tammo Hinrichs (Original 6510 and SID routine) and
Rainer Sinsch (linux engine)

## The story of tinysid-linux.tar.gzip file

2013-04:
I downloaded TinySID at http://www.rsinsch.de/data/files/ce4f22308f/tinysid-linux.tar.gzip -> tinysid-linux.tar.gz  
sha1sum: 31edf36cd9e0658c8286e4a3a97901a8e30ba0ca

2013-08-15:
I put tinysid at https://www.linuxunderground.be/distfiles/tinysid-linux.tar.gz

2015-03-26:
I noticed that tinysid-linux had been removed from http://www.rsinsch.de  
However, it was still available at http://www-stud.rbi.informatik.uni-frankfurt.de/~sinsch/new/tinysid-Dateien/downloads/tinysid-linux.tar.gzip

2026-05-18:
Original source code is now available with a [stable URL via archive.org](https://web.archive.org/web/20260518184951/https://www.linuxunderground.be/distfiles/tinysid-linux.tar.gz)

2026-06-26:
Creating a github repository
